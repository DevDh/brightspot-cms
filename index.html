Brightspot CMS is a open source Content Management System built on top of the Dari Framework. Each instance is tailored to the application it powers, as it is automatically generated from the objects it handles. Brightspot CMS is uniquely adaptable to change, and delivers an easy-to-use and familiar editorial experience. Download Brightspot CMS »
