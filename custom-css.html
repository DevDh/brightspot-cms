Brightspot now allows custom CSS styles to be added as options within the Rich Text toolbar. To enable custom options for editors, add CSS styles using the CMS Settings page interface (Admin > Settings) :



Create a name for the menu item, and then specify the HTML tag required. Above we have used a span class, however you could use div, p or a h1 tag, for example.
