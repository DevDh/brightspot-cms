Start by finding the bulk upload widget on the Brightspot dashboard. Select the content type that you wish to upload.

Once the content type is selected, you can either drag and drop the files from your desktop or select multiple files from the dropdown.

Once your files have been uploaded the can all be found in the drafts widget on the dashboard. There is also an option to save the image with the actual file name.
