External Content, such as Tweets, Videos or Photos can now be added to a Rich Text area by adding an enhancement, and pasting the content URL.



To embed external content click 'Add Enhancement' in the Rich Text toolbar. In the pop-up window use the Create drop-down to find and add a new External Content object. Paste the URL for the content you would like to embed. Examples include YouTube videos, Tweets, Flickr images, Vimeo videos and Vine and Instagram posts. You can preview your content before saving.



You can provide sizes for exact height and width of the embedded content, or leave blank to use the default sizes.

Summary: External Content

Brightspot uses the oEmbed protocol to make it easy to add external media to your content.
