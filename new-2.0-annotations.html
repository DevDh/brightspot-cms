New annotations are available in Brightspot 2.0. You can see the full documentation for annotations online but here are a few highlights:

Dropdown

Use the @ToolUi.DropDown annotation to turn a short set of returned results into a dropdown list instead of the more robust pop-up search. This is especially helpful if there are only a few results available.

Global Filtering

Dashboard widgets now offer custom filters for editors. Using the @ToolUi.GlobalFilter you can add any class as a global filter. This annotation is most helpful when there is a global category, like Author, that is used across multiple content types.

Sorting / Filtering

Our new search improvements allow editors to sort and filter their results with greater precision. Use the two new annotations, @ToolUi.Filterable and @ToolUi.Sortable, to specify fields within an object that can be sorted or filtered. Do your editors want the ability to sort articles by total page views? Add this annotation to the field to give them this kind of option. (Note: These fields must also be @Indexed).

Improvements in search, combined with custom filtering, allow developers and editors to work together to make finding content easier. Working with your editorial team to define new sorting and filtering options will unlock the potential of these powerful new features for them.
