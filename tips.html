Editorial Tip - Historic Snapshots

Brightspot not only stores each and every published version of content, but also allows you to create snapshots of past versions, so you can save and reuse. This tip walks through how to create your own historic snapshots of content. Read the full Historic Snapshots tip

Editorial Tip - Inline Editing

Brightspot 2.0 provides a new and improved inline editor tool. Formerly known as the ‘bookmarklet’ this tool allows editors to access and edit their content from the front end of their site, with a fully functional interface for editing text, images or SEO. In this tip find out how to access the tool and make edits to your site content directly inline. Read the full Inline Editing tip.
